# Python Coding Introduction - Starter Code

Welcome to your first Python programming mission!

## Getting Started

1. Fork this repository on GitHub
2. Clone to your computer
3. Open in VS Code
4. Open the integrated terminal (View â†’ Terminal or Ctrl+`)

## Files in This Project

- `hello.py` - Your first Python file (already complete)
- `adventure.py` - Your text adventure template (you'll build this)

## Running Your Code

In the terminal, type:

```bash
python hello.py
```

You should see a message! This means Python is working.

**Note:** On some systems, you may need to use `python3` instead of `python`.

## Your Mission

Open `adventure.py` and follow the TODO comments to create your text adventure.

Run your adventure anytime with:

```bash
python adventure.py
```
